## üìñ About

**BazaarMind** bridges the gap between traditional commerce and digital intelligence for the unorganized retail sector in Bangladesh. Unlike cold, corporate analytics tools, BazaarMind is designed to feel like a trusted business partner ("Munshiji")‚Äîwarm, intuitive, and collaborative.

### Design Philosophy

- **Warm & Human:** Interfaces designed with empathy, avoiding data overload.
- **SME-Focused:** Tailored workflows for small traders, wholesalers, and shopkeepers.
- **AI as Partner:** Recommendations are presented as collaborative advice, not robotic commands.
- **Narrative Layout:** Information flows as a story (Pulse ‚Üí Action ‚Üí Alert), not just a grid of metrics.
- **Hyper-Localized:** Authentic typography (Hind Siliguri) and Bengali-first context.

## ‚ö° Getting Started

Follow these steps to set up the project locally.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1.  **Clone the repository**

    ```bash
    git clone [https://github.com/yourusername/bazaar-mind.git](https://github.com/yourusername/bazaar-mind.git)
    cd bazaar-mind
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

The dashboard will launch automatically at `http://localhost:3000`.

### Building for Production

To create an optimized production build:

```bash
npm run build
```
