## Design Philosophy

- **Warm & Human**: Feels like a trusted business partner, not a cold corporate tool
- **SME-Focused**: Built for small shop owners, traders, and wholesalers
- **AI as Partner**: AI suggestions feel collaborative, not robotic
- **Storytelling Layout**: Information presented as a narrative, not just metric tiles
- **Bangladesh-Centric**: Uses Bengali language and local business context

## Technology Stack

- **React 18** - Modern component-based UI
- **Vite** - Fast build tool and dev server
- **Custom CSS** - No Tailwind, no Material UI - unique design
- **Bengali Fonts** - Hind Siliguri for authentic typography

## Design Principles

1. **Color Palette**: Warm tones inspired by Bangladesh markets
   - Primary Warmth: `#FF6B35`
   - Primary Trust: `#004E89`
   - Secondary Gold: `#F7B801`
   - Accent Green: `#06A77D`

2. **Typography**: Bengali-first with Hind Siliguri font

3. **Layout**: Three-column storytelling layout
   - Left: What's happening today
   - Center: What AI suggests
   - Right: What needs attention

4. **Interactions**: Smooth, meaningful animations that feel natural
